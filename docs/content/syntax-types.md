# Types

An overview of common types and annotations in Steel.

## Primitives

```steel
let a: int = 1;
let b: float = 3.14;
let c: bool = true;
let s: string = "steel";
```

## Arrays

```steel
let nums: int[] = [1,2,3];
```

## Structs

```steel
struct Point { x: int, y: int }
let p = Point { x: 1, y: 2 };
```
