# Install Steel

Install the Steel toolchain and verify your setup.

## Requirements

- Windows, macOS, or Linux
- Recent terminal and a code editor

## Installation

Head over to the [github releases](https://github.com/swzldev/Steel/releases) page and download the latest release for your platform.

Make sure to add it to your system PATH.

## Verify

```bash
steel --version
```

You should see a version like `steelc version 0.4.0`.
