# Functions

Functions are reusable, named blocks of code.

## Declaring functions

The syntax for declaring a function is as follows:

```
func <function-name>(<optional-parameters>) -> <return-type> {
  // Function code goes here...
}
```

For example, here is the declaration for a basic `add` function:

```
func add(a: int, b: int) -> int {
    return a + b;
}
```

- **add**: The name of the function - used to call it.
- **a**: The first parameter of the function, named **a** and of type **int**.
- **b**: The second parameter of the function, named **b** and of type **int**.
- **-> int**: The return type of the function, in this case **int**.
